# 🛍️ E-Commerce Web Application

A fully customizable **E-Commerce Web Application** built with **Laravel, Blade, Livewire, and Tailwind CSS**.

This platform delivers a modern shopping experience for customers and a powerful admin dashboard that gives full control over products, orders, revenue, and website customization.

---

## 🚀 Features

### 🛒 Customer Features

- Browse products by categories
- Product details page
- Add to cart & manage cart
- Apply coupons & discounts
- Secure checkout process
- Order tracking
- Fully responsive modern UI

---

### ⚙️ Admin Dashboard

A powerful and fully customizable admin panel designed for complete business management.

#### 📦 Product Management
- Create, edit, and delete products
- Upload and manage product images
- Stock management system
- Category management

#### 📊 Order Management
- View and manage all orders
- Update order status
- Track revenue and sales performance
- Manage delivery prices

#### 💰 Discounts & Coupons
- Create and manage coupon codes
- Set expiration dates
- Control usage limits
- Apply percentage or fixed discounts

#### 🎨 Full Website Customization
- Change website themes
- Customize primary and secondary colors
- Upload and update logos
- Modify branding elements
- Personalize layout sections

---

## 🧰 Tech Stack

- **Backend:** Laravel  
- **Frontend:** Blade  
- **Reactive Components:** Livewire  
- **Styling:** Tailwind CSS  
- **Database:** MySQL
